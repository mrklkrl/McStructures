◆ McSinglyLinkedList

typedef struct McSinglyLinkedList McSinglyLinkedList

McSinglyLinkedList - a Singly Linked List.

This type of list can be traversed in a single direction only using an iterate function Elements can be appended, erased, inserted, or found. Must be initialized with mcSinglyLinkedListInit();

◆ McSinglyLinkedListElement

typedef struct McSinglyLinkedListElement McSinglyLinkedListElement

McSinglyLinkedListElement - an element belonging to a McSinglyLinkedList.

This type of element stores a pointer to the next element in the list.
